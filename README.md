# Cryptbound

A mobile-first, deterministic 100-level first-person dungeon crawler built for Vercel.

## Included
- 100 reproducible procedural levels across 10 themes
- Boss floor every 10 levels
- Warrior and Mage classes
- Melee and magic combat
- Spiders, skeletons, wraiths, and bosses
- Loot relics, gold, XP, and permanent upgrades
- Touch D-pad, attack/spell buttons, swipe-to-look
- Keyboard controls: WASD, arrows, Space, E
- Local autosave and unlocked-level selection
- Responsive landscape HUD and installable web-app manifest

## Run
```bash
npm install
npm run dev
```

## Deploy to Vercel
1. Upload this folder to a GitHub repository.
2. In Vercel, choose **Add New → Project**.
3. Import the repository.
4. Leave Framework Preset as **Next.js**.
5. Click **Deploy**.

No environment variables or database are required.
